# ph-area-code

[![npm version](https://img.shields.io/npm/v/ph-area-code.svg)](https://www.npmjs.com/package/ph-area-code)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Philippine telephone area code lookup and validation

## Installation

```bash
npm install ph-area-code
```

## Functions

- `validateAreaCode` — Validate Philippine area code
- `getAreaName` — Get area name from telephone code
